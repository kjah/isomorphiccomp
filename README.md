
#Isomorphic Component Model

基于EJS模板的,前后端同构的,页面及组件模型,其中包含该一个简易的设计器示例.

###运行
```
npm i
npm run start
```

###网站前台：<br>
http://localhost:3000/index.html


###简易设计器：<br>
http://localhost:3000/designer/html/index.html

鼠标移动到对应组件选中可编辑组件参数，并即时显示修改结果。


###此示例使用包清单：<br>
Koa2    ---示例用web服务器<br>
ejs     ---同构模板技术<br>
cheerio ---DOM解析<br>
mockjs  ---模拟数据<br>
